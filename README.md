## Quick Start

```bash
# Git clone
git clone https://github.com/alphaolomi/chat-in-mongo

# Copy 
cp .env.example .env
 
# Install dependencies
npm install

# Serve on localhost:3000
npm run dev

# Run for production
npm start

# Run tests
npm run test
```
